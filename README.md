# 🪓 Tree Capitator

A lightweight and highly customizable Minecraft plugin that revolutionizes woodcutting. Instead of breaking every block individually, this plugin allows you to fell entire trees with a single blow – efficient, performant, and server-friendly.

## 🌟 Features
- **Intelligent Recursion**: Automatically detects connected log structures.
- **Performance Optimized**: Adjustable limits for maximum block counts to prevent server lag.
- **Survival-Ready**: Respects axe durability and supports enchantments like Unbreaking.
- **Personalized**: Every player can toggle the feature for themselves using `/treecap toggle`.
